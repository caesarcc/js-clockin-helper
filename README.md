# js-clockin-helper

Calculadora de Jornada de Trabalho — PWA simples construída com Vite + React + TypeScript + Tailwind.

Getting started

Install dependencies:

```bash
npm ci
```

Run dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run unit tests:

```bash
npm run test:unit
```
